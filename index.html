Request Header Parser Microservice

Get the IP address, language and operating system of a browser.


Example usage:

https://request-header-parser-microservice-rohanimmanuel.c9users.io/whoami

    "language":  "en-US",
    "software":  "Windows NT 6.1; WOW64"
}
